# React Boilerplate
React boilerplate with webpack, typescript, eslint, jest, react-testing-library

## Features
* Webpack 4
* Eslint
* Sass

## Quick start
Clone the repo

    git clone https://github.com/shaminmeerankutty/React-Starter.git

Install node modules

    npm install

Run development server

    npm start

Build files

    npm run build
